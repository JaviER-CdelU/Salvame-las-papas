# Sálvame las Papas — PWA con instalación visible

Esta versión incluye una sección visible en `index.html` con la Súper Papa y el botón **Instalar la aplicación**.

## Para subir a GitHub Pages

1. Descomprimí este ZIP.
2. Subí TODO el contenido a la raíz del repositorio.
3. Conservá la carpeta `icons`.
4. En GitHub: Settings → Pages → Deploy from a branch → main → /root.
5. Abrí la URL publicada con HTTPS.

## Cómo se instala

- Android/Chrome: tocar “Instalar la aplicación”.
- iPhone/Safari: Compartir → Agregar a pantalla de inicio.
- PC/Chrome o Edge: ícono de instalación en la barra de direcciones.

Al abrir `index.html` directamente desde la computadora, el navegador puede no ofrecer instalarla. La PWA debe estar publicada con HTTPS o probarse en localhost.
