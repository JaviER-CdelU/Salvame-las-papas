🧉 Diccionario entrerriano

Palabras, expresiones y costumbres que forman parte de nuestra manera de hablar. Algunas son tradicionales, otras informales y otras deben usarse con cuidado.

98 vocablos disponibles
A Uso informal

Abatatarse

Avergonzarse, ponerse nervioso o no saber cómo reaccionar.

Ejemplo: “Se abatato cuando tuvo que hablar delante de todos.”

A Uso regional

Allí

Forma coloquial de señalar que algo queda cerca o ahí nomás.

Ejemplo: “La despensa queda allí, cruzando la calle.”

A Uso informal

Achacar

Culpar o responsabilizar a alguien por algo.

Ejemplo: “A mí no me achaqués, que yo no tuve nada que ver.”

A Uso vulgar

A todo culo

Expresión para decir que algo se hizo con mucho lujo, despliegue o gasto.

Ejemplo: “Abrieron el bar nuevo a todo culo.”

A Uso informal

Angurriento

Persona ansiosa por comer, obtener o quedarse con más de lo que le corresponde.

Ejemplo: “No seas angurriento, dejá un poco para los demás.”

A Uso informal

Apestado

Muy enfermo, engripado o con fuerte malestar.

Ejemplo: “Hoy no salgo, estoy apestado.”

A Uso informal

Apichonado

Triste, desanimado o bajoneado.

Ejemplo: “Después del partido quedó apichonado.”

A Uso regional

Aprontar

Preparar algo para que quede listo.

Ejemplo: “Aprontá las cosas, que en diez minutos nos vamos.”

A Uso vulgar / revisar contexto

Afrechudo

Persona con mucha energía o excitación acumulada.

Ejemplo: “Mirá cómo corre ese afrechudo.”

A Uso cotidiano

Almacén / Despensa

Comercio pequeño de barrio donde se venden alimentos y productos cotidianos.

Ejemplo: “Voy al almacén a comprar yerba y pan.”

B Uso informal

Bolacero

Persona que miente, exagera o inventa historias difíciles de creer.

Ejemplo: “No le creas todo, ese gurí es medio bolacero.”

B Uso informal

Bolazo

Mentira, exageración o historia poco creíble.

Ejemplo: “Manso bolazo me acabás de decir.”

B Gastronomía

Butifarra

Embutido preparado generalmente con carne de cerdo y condimentos.

Ejemplo: “Para la picada compramos queso y butifarra.”

B Uso informal

Bajo el agua

Expresión para indicar una situación económica muy difícil. También se pronuncia “bajolagua”.

Ejemplo: “Este mes no puedo salir, estoy bajolagua.”

B Panadería

Bollo

Factura redonda, muchas veces con crema pastelera.

Ejemplo: “Traeme dos bollos para el mate.”

B Uso despectivo

Bagallo

Persona considerada poco atractiva.

Ejemplo: “Ese comentario fue desubicado; no da decirle bagallo a nadie.”

C Uso cotidiano

Caballito

Forma de llevar a una persona sobre la espalda.

Ejemplo: “El gurí volvió a casa a caballito.”

C Uso informal

Caldero

Persona tacaña o poco dispuesta a compartir.

Ejemplo: “No seas caldero, convidá un poco.”

C Uso despectivo / revisar

Calzonudo

Persona considerada débil de carácter o muy dominada por su pareja.

Ejemplo: “Le dijeron calzonudo porque nunca decide nada solo.”

C Uso regional

Cachiquengue

Desorden, lío o situación caótica.

Ejemplo: “Ordená ese cachiquengue antes de que llegue la gente.”

C Doble significado

Carpir / salir carpiendo

Carpir es cortar malezas; salir carpiendo es huir o irse rápidamente.

Ejemplo: “Cuando vio venir la tormenta, salió carpiendo.”

C Uso informal

Catinga

Olor fuerte y desagradable.

Ejemplo: “¡Qué catinga quedó después de limpiar el pescado!”

C Uso regional

Chinguiado

Torcido, desviado o mal alineado.

Ejemplo: “El estante quedó medio chinguiado.”

C Clima

Chispear

Llover muy suavemente.

Ejemplo: “Todavía no llueve fuerte, está chispeando nomás.”

C Gastronomía

Churrasquera

Lugar preparado para cocinar carne a las brasas.

Ejemplo: “Prendé el fuego en la churrasquera.”

C Uso infantil / advertencia

Cocoño

Castigo o juego brusco que consiste en dar golpes con los nudillos en la cabeza.

Ejemplo: “Al que pierde le hacen cocoño.”

C Uso vulgar

Cara de guasca

Insulto leve o burlón, generalmente entre personas de confianza.

Ejemplo: “¡Dejá de molestar, careguasca!”

C Uso informal

Camorra

Conflicto buscado de manera intencional para provocar una pelea.

Ejemplo: “Dejá de buscar camorra.”

C Objeto cotidiano

Calorito

Calentador eléctrico utilizado para calentar agua dentro del termo.

Ejemplo: “Cuidado con prender el calorito fuera del agua.”

C Uso informal

Camorrero

Persona que busca peleas o conflictos.

Ejemplo: “Ese muchacho es muy camorrero.”

C Uso regional

Colero

Gomita para atarse el pelo.

Ejemplo: “Prestame un colero.”

C Gastronomía

Carlitos

Sándwich caliente de jamón y queso.

Ejemplo: “Pedimos dos carlitos y una gaseosa.”

C Uso regional

Chijete

Salida o movimiento muy rápido.

Ejemplo: “Salió como chijete.”

C Uso informal

Cuerear

Hablar mal de una persona cuando no está presente.

Ejemplo: “Dejá de cuerear a los demás.”

C Uso vulgar

Cursiadera

Diarrea o malestar intestinal fuerte.

Ejemplo: “Me agarró una cursiadera en el viaje.”

C Uso regional

Cluenque

Mueble u objeto desnivelado o inestable.

Ejemplo: “Esta mesa está en cluenque.”

C Uso informal

Clinudo

Persona de pelo largo.

Ejemplo: “El gurí clinudo toca en una banda.”

C Uso expresivo

Chico / Manso / Flor

Palabras usadas para intensificar y expresar que algo es grande o notable.

Ejemplo: “¡Manso auto! / Flor de problema.”

C Uso informal

Cuentero

Persona que inventa historias y las presenta como verdaderas.

Ejemplo: “Dejá de decir bolazos, sos re cuentero.”

C Uso positivo

Corajudo

Valiente, decidido o esforzado.

Ejemplo: “Fue corajudo y se animó a intentarlo.”

C Gastronomía

Chuleta

Bife con hueso.

Ejemplo: “Compramos unas chuletas para la parrilla.”

C Uso sexual / revisar

Cachonda

Persona excitada o con actitud provocativa.

Ejemplo: “La palabra puede resultar incómoda fuera de ámbitos de confianza.”

C Uso informal

Caché

Expresión para indicar que se descubrió a alguien haciendo algo.

Ejemplo: “¡Ah, te caché sacando fotos!”

C Uso informal

Chito

Orden para pedir silencio.

Ejemplo: “Chito, no digas nada todavía.”

C Uso cotidiano

Cobija

Frazada o manta para abrigarse.

Ejemplo: “Alcanzame una cobija, hace frío.”

D Uso informal

Despelote

Gran desorden, confusión o conflicto.

Ejemplo: “Quedó un despelote después de la fiesta.”

D Uso informal

Descuajeringar

Desarmar, romper o dejar algo muy deteriorado.

Ejemplo: “Se descuajeringó la bicicleta.”

D Uso regional

Desmayau

Persona distraída, desorientada o que suele estar en otra.

Ejemplo: “Ese gurí es un desmayau.”

D Uso humorístico

Dificilongo

Algo especialmente difícil o complicado.

Ejemplo: “El trámite está dificilongo.”

D Uso regional

Disparada

Con mucha rapidez o urgencia.

Ejemplo: “Salió de disparada para el trabajo.”

D Uso cotidiano

Despelechar

Perder o sacar la piel superficial, especialmente después de tomar sol.

Ejemplo: “Me estoy despelechando todo.”

E Uso regional

Elastiquín

Gomita elástica para cerrar bolsas o sujetar objetos.

Ejemplo: “Poné un elastiquín al paquete.”

E Referencia local

El Parque

Forma local de referirse a un parque conocido de la ciudad.

Ejemplo: “Nos encontramos en El Parque.”

E Uso regional

Espores

Zapatillas deportivas. Deriva humorísticamente de “sport shoes”.

Ejemplo: “Me compré unos espores nuevos.”

E Uso tradicional

Esquela

Nota breve o comunicación familiar para avisar algo.

Ejemplo: “Dejale una esquela diciendo dónde fuimos.”

E Uso rioplatense

Empilcharse

Vestirse bien o con ropa elegante.

Ejemplo: “Se empilchó para la fiesta.”

E Uso informal

Encularse / Enchincharse

Enojarse o quedar muy molesto.

Ejemplo: “Se enchinchó porque no lo invitaron.”

E Uso regional

Engañapichanga

Engaño, mentira o maniobra poco seria.

Ejemplo: “Eso fue una engañapichanga.”

E Uso informal

Enyetado

Con mala suerte o atravesando una racha negativa.

Ejemplo: “Últimamente ando medio enyetado.”

F Uso informal

Fiero

Muy feo, desagradable o difícil.

Ejemplo: “Está fiero el tiempo.”

F Uso informal

Forrado

Persona con mucho dinero.

Ejemplo: “Dicen que está forrado en plata.”

G Panadería

Galleta

Pieza de panadería cuadrada, parecida a un librito.

Ejemplo: “Compramos galletas para el mate.”

G Uso regional

Gurí / Gurisa

Niño o niña.

Ejemplo: “Ese gurí juega a la pelota todo el día.”

G Variante regional

Gollete / Goyete

Sentido o lógica. Se usa en la frase “no tiene gollete/goyete”.

Ejemplo: “Lo que hizo no tiene gollete.”

G Uso informal

Guachazo

Golpe fuerte y repentino.

Ejemplo: “Me pegué un guachazo con la ventana.”

G Uso cotidiano

Girasoles

Semillas de girasol para comer.

Ejemplo: “Compramos maní y girasoles.”

G Uso informal

Guarangada / Guasada

Acción grosera, tosca o de mala educación.

Ejemplo: “Lo que dijo fue una guasada.”

G Uso despectivo

Guampudo / Guampuda

Persona a la que su pareja le fue infiel.

Ejemplo: “La expresión puede ser humillante.”

H Orientación

Haciendo cruz

Expresión para indicar que un lugar está enfrente y un poco hacia un costado.

Ejemplo: “La farmacia queda haciendo cruz con la plaza.”

H Uso informal

Hilacha

Se usa en “mostrar la hilacha”: dejar ver el verdadero carácter o la falta de educación.

Ejemplo: “Ahí mostró la hilacha.”

H Uso regional

Hurguete

Persona curiosa que revisa o averigua todo.

Ejemplo: “No seas hurguete.”

J Uso tradicional

Julepe

Miedo o susto repentino.

Ejemplo: “Qué julepe me pegué cuando ladró el perro.”

M Uso familiar

Mami

Forma afectuosa y habitual de llamar a la madre.

Ejemplo: “Mami, ¿dónde dejaste el mate?”

M Uso regional

Mandale guacha nomás

Expresión para alentar a hacer algo de inmediato y con decisión.

Ejemplo: “¿Me anoto? Mandale guacha nomás.”

M Uso informal

Mamado

Borracho o muy alcoholizado.

Ejemplo: “Volvió mamado de la fiesta.”

M Pronunciación regional

Mavále

Contracción enfática de “más vale”.

Ejemplo: “¿Vamos? ¡Mavále!”

M Costumbre regional

Mate de té

Infusión preparada en jarrito con té, hierbas, limón, azúcar y agua caliente.

Ejemplo: “Prepararon un mate de té bien dulce.”

N Uso regional

No tiene goyete

Frase para indicar que algo no tiene sentido o lógica.

Ejemplo: “Che, eso no tiene goyete.”

P Uso regional

Pandorga

Barrilete o cometa.

Ejemplo: “Los gurises remontaron una pandorga.”

P Uso vulgar

Pajero

Insulto o llamado de atención muy frecuente en contextos informales. Su sentido depende del tono.

Ejemplo: “¡Che, pajero, te olvidaste las llaves!”

P Uso familiar

Papi

Forma afectuosa y habitual de llamar al padre.

Ejemplo: “Papi, ¿me llevás?”

P Juego tradicional

Payanga

Juego tradicional realizado con cinco piedras pequeñas.

Ejemplo: “Jugábamos a la payanga en la vereda.”

P Gastronomía

Pororó

Pochoclo o palomitas de maíz.

Ejemplo: “Hicimos pororó para mirar una película.”

P Bebida alcohólica

Porrón

Botella pequeña de cerveza.

Ejemplo: “Pidieron un porrón bien frío.”

P Uso humorístico

Pijerto

Pieza, objeto o elemento pequeño cuya función no se conoce bien.

Ejemplo: “¿Para qué sirve este pijerto?”

P Uso tradicional

Pitín

Persona joven encargada de hacer mandados.

Ejemplo: “Mandaron al pitín a comprar pan.”

P Carnicería

Pulpa / Pulpa especial

Corte de carne considerado tierno y de buena calidad.

Ejemplo: “Compramos pulpa para hacer milanesas.”

R Uso regional

Relajar

Hacer bromas o cargar a alguien.

Ejemplo: “Lo estaban relajando por el corte de pelo.”

R Uso regional

Reparar

Copiar o imitar una acción.

Ejemplo: “La maestra dijo que dejara de reparar al compañero.”

R Clima

Refucilo

Relámpago.

Ejemplo: “Se vio un refucilo antes de la tormenta.”

R Uso informal

Repugnante

Persona soberbia, peleadora o desagradable. A veces se pronuncia “repunante”.

Ejemplo: “No seas repugnante con la gente.”

S Gastronomía

Sancocho

Comida quemada o demasiado cocida.

Ejemplo: “Se me pasó la comida y quedó sancochada.”

S Folclore

Solapa

Personaje del folclore entrerriano que, según el mito, aparece durante la siesta.

Ejemplo: “De chico le tenían miedo a la Solapa.”

S Clima

Sereno

Rocío o humedad que cae durante la noche.

Ejemplo: “Está cayendo sereno.”

S Juego tradicional / advertencia

Soplamoco

Juguete casero hecho con un globo y un tubo para disparar pequeñas bolitas.

Ejemplo: “Jugaban con un soplamoco.”

T Bebida

Tereré

Mate de yerba preparado con agua fría, jugo u otros líquidos.

Ejemplo: “Tomamos tereré porque hacía mucho calor.”

V Uso tradicional

Vaquero

Pantalón de jean.

Ejemplo: “Me compré un vaquero nuevo.”

V Uso vulgar

Verga

Palabra vulgar usada como exclamación, insulto o para expresar enojo, mala calidad o frustración.

Ejemplo: “¡Pero qué verga, se cortó la luz!”

Nota editorial: las expresiones marcadas como vulgares, despectivas o sensibles se conservan con una advertencia de uso. No representan recomendaciones para tratar a otras personas.
